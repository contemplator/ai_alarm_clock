## 1. 設計原則

### 1.1 核心理念

- **簡潔直覺**：鬧鐘 App 需要快速設定，不需複雜操作
- **iOS 原生風格**：遵循 iOS Human Interface Guidelines
- **暗色優先**：鬧鐘常在睡前/起床時使用，暗色保護眼睛
- **大字體**：時間顯示要醒目，便於快速瀏覽

### 1.2 設計風格

- 參考 iOS 原生時鐘 App 風格
- 使用 Cupertino 風格元件
- 圓角卡片設計
- 漸層與毛玻璃效果增添質感

## 6. 動畫規格

### 6.1 頁面轉場

- 使用 iOS 預設 Cupertino 風格轉場
- 持續時間：300ms
- 曲線：`Curves.easeInOut`

### 6.2 元件動畫

| 動畫類型     | 持續時間 | 曲線          |
| ------------ | -------- | ------------- |
| Switch 切換  | 200ms    | easeInOut     |
| 卡片按下效果 | 100ms    | easeOut       |
| 列表項目進入 | 300ms    | easeOutCubic  |
| 刪除動畫     | 250ms    | easeInBack    |
| Loading 旋轉 | 1000ms   | linear (loop) |

### 6.3 鬧鐘響起頁動畫

- 時間數字：脈動效果 (scale 1.0 ↔ 1.02，時間 2000ms)
- 停止按鈕：呼吸燈效果 (opacity 0.7 ↔ 1.0)
- 新聞文字：淡入效果

## 10. 使用者流程

### 10.1 首次使用流程

```
App 啟動
    │
    ▼
請求通知權限
    │
    ├── 允許 ──▶ 進入首頁 (空鬧鐘列表)
    │
    └── 拒絕 ──▶ 顯示說明對話框
                  │
                  ▼
              引導至設定開啟權限
```

### 10.2 設定鬧鐘流程

```
首頁點擊「➕」
    │
    ▼
顯示新增鬧鐘頁
    │
    ▼
選擇時間
    │
    ▼
選擇重複週期
    │
    ▼
(可選) 設定標籤、鈴聲、新聞來源
    │
    ▼
點擊「儲存」
    │
    ▼
返回首頁，顯示新鬧鐘
```

### 10.3 鬧鐘觸發流程

```
系統通知觸發 (播放鈴聲)
    │
    ▼
使用者點擊通知
    │
    ▼
App 開啟，顯示鬧鐘響起頁
    │
    ├── 等待 15 秒 ──▶ 自動開始播報新聞
    │
    └── 點擊「停止」──▶ 結束，返回首頁

新聞播報中
    │
    ├── 播報完畢 ──▶ 顯示完成提示，返回首頁
    │
    └── 點擊「停止」──▶ 中斷播報，返回首頁
```
